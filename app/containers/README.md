###容器组件（智能组件）
<p>连接数据，修改数据的组件</p>
<ul>
    <li>生成方式：
        <ul>
            <li>使用ES6 class 生成；</li>
            <li>使用高阶组件 生成，如：redux connect</li>
        </ul>
    </li>
    <li>数据传输方式：
        <ul>
            <li>使用 本地储存、ajax等数据请求、redux connect。</li>
        </ul>
    </li>
</ul>
<p>目录结构：</p>
<ul>
    <li>
        一级目录：对应pages文件
        <ul>
            <li>二级目录：pages内组件</li>
        </ul>
    </li>
</ul>
